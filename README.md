# 👗 Tiara Mode Kebaya

Aplikasi Flutter untuk menampilkan jasa penjahitan **Tiara Mode** — spesialis kebaya, jahitan halus, dan layanan vermak di Parung Panjang.

---

## 📱 Tampilan Awal

- Logo dan nama brand di AppBar
- Tombol navigasi cepat:
  - Instagram Portfolio
  - Panduan Ukur Badan
- Gambar banner kebaya
- Bottom navigation:
  - Home
  - Portfolio
  - Layanan
  - Pesan
